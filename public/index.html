API

Simple, user-friendly API documentation for developers, students, and anyone building Islamic finance tools.

Why This API Is Easy To Use

  • Start quickly with one `GET` endpoint and one `POST` endpoint.
  • Get live Nisab values with currency conversion.
  • Calculate Zakat with smart insights, not only a number.
  • Supports both `gold` and `silver` Nisab basis.
  • Updated with 50 supported currencies.

Endpoints

GET https://zakat-calculator-coral-one.vercel.app/api/nisab?currency=USD
POST https://zakat-calculator-coral-one.vercel.app/api/calculate

Quick Start

  • Use `GET /api/nisab` to show the current Nisab threshold.
  • Send assets and liabilities to `POST /api/calculate`.
  • Read `calculation`, `nisab`, and `insights` from the JSON response.

Best For Users

  • Zakat calculator websites
  • Islamic finance mobile apps
  • Educational demos and student projects
  • Backend services needing simple JSON integration
