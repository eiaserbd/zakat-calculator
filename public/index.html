🔌 API Documentation

এই API ব্যবহার করে আপনার জাকাত হিসাব স্বয়ংক্রিয়ভাবে গণনা করুন

📡 API Endpoint:
POST /api/calculate
Full URL:
📝 Request Format:
Content-Type: application/json

📊 বিভিন্ন পরিস্থিতির উদাহরণ:

সাধারণ হিসাব:
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"assets":{"cash":500000,"gold":100000,"silver":50000,"savings":200000,"business":300000},"liabilities":{"debts":50000,"expenses":10000},"currency":"BDT"}'
শুধুমাত্র নগদ অর্থ:
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"assets":{"cash":1000000},"liabilities":{},"currency":"BDT"}'
স্বর্ণ ও রৌপ্য সহ:
curl -X POST \
  -H "Content-Type: application/json" \
  -d '{"assets":{"gold":200000,"silver":100000},"liabilities":{"debts":25000},"currency":"BDT"}'

📋 Request Schema:

{
  "assets": {
    "cash": number,      // নগদ অর্থ
    "gold": number,      // স্বর্ণের মূল্য
    "silver": number,    // রৌপ্যের মূল্য
    "savings": number,   // সঞ্চয়
    "business": number   // ব্যবসায়িক সম্পদ
  },
  "liabilities": {
    "debts": number,     // ঋণ
    "expenses": number  // মাসিক খরচ
  },
  "currency": "BDT"     // মুদ্রা (ঐচ্ছিক)
}

📋 Response Schema:

{
  "success": true,
  "calculation": {
    "total_assets": number,
    "total_liabilities": number,
    "net_wealth": number,
    "nisab_threshold": number,
    "is_zakat_applicable": boolean,
    "zakat_amount": number,
    "zakat_percentage": 2.5,
    "currency": "BDT"
  }
}
