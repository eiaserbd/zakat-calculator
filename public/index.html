API Overview

HTTP Method URL Description Parameters
GET /api/nisab?currency=USD Returns current Nisab values for gold and silver in the selected currency. currency
POST /api/calculate Calculates Zakat using assets, liabilities, currency, and Nisab basis. assets, liabilities, currency, nisab_basis
GET Nisab endpoint

Use this endpoint to fetch current Nisab values for gold and silver in the currency you choose.

GET https://zakat-calculator-coral-one.vercel.app/api/nisab?currency=USD
POST Calculation endpoint

Send assets, liabilities, currency, and Nisab basis to calculate Zakat and receive a clear response.

POST https://zakat-calculator-coral-one.vercel.app/api/calculate Content-Type: application/json

What To Send

  • Use `currency` such as `BDT`, `USD`, `EUR`, or any supported currency.
  • Use `nisab_basis` as `gold` or `silver`.
  • Put eligible wealth items inside `assets`.
  • Put deductible obligations inside `liabilities`.

What You Receive

  • Total assets, total liabilities, and net wealth
  • Selected Nisab threshold
  • Whether Zakat is applicable
  • Zakat amount and smart insights
  • Support for 157 currencies from the Nisab API

Example POST Request Body

{ "assets": { "cash": 500000, "gold": 125000, "savings": 220000 }, "liabilities": { "debts": 70000, "expenses": 25000 }, "currency": "BDT", "nisab_basis": "gold" }

Example Response

{ "success": true, "currency": "BDT", "nisab_basis": "gold", "calculation": { "total_assets": 845000, "total_liabilities": 95000, "net_wealth": 750000, "nisab_threshold": 743580, "is_zakat_applicable": true, "zakat_amount": 18750, "zakat_percentage": 2.5 }, "insights": [ { "type": "success", "title": "Zakat is due", "message": "Your payable Zakat is 18,750 BDT at 2.5% of net eligible wealth." } ] }

Important Request Fields

assets.cash
Cash in hand, bank balance, or any money immediately available.
assets.gold
Current market value of the gold you own.
assets.silver
Current market value of the silver you own.
assets.savings
Savings, deposits, or money reserved for later use.
liabilities.debts
Short-term debts or obligations that can be deducted.
currency
Any supported currency returned by the API.
nisab_basis
Choose `gold` or `silver` depending on the Nisab threshold you want to apply.
